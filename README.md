# Kashmir Shaivism Texts

Plain-English audiobook renderings of the foundational texts of non-dual Kashmir Shaivism (Trika).

These are written to be **read aloud** — from a teleprompter, into a microphone — in clear, modern English. They use everyday metaphors a contemporary listener can follow, and they deliberately avoid the dense brackets, diacritics, and technical apparatus of scholarly translations. The goal is faithfulness to the meaning, set free for the ear.

These renderings are **not** translations. They do not reproduce any existing translator's text. They are fresh prose retellings of the underlying teachings, written from the ancient (public-domain) source verses so the ideas can be heard and understood by a general modern audience. Each text closes with a fidelity note describing how closely it tracks the original and what, if anything, was compressed for listening.

## How to read the collection

A natural sequence for a newcomer:

1. **Heart of Recognition** — the whole system in twenty short steps; the best doorway.
2. **Shiva Sutra** — the root aphorisms the tradition is built on.
3. **Spanda Karika** — the pulse of consciousness; the vibration teaching.
4. **Vijnana Bhairava Tantra** — the practice manual: 112 meditation techniques you can actually do, with cross-references to the Tantrasara.
5. **Tantrasara** — the full system: paths, cosmos, and the rituals of initiation and worship.

## A note on the tradition

Non-dual Kashmir Shaivism holds that there is only one reality — pure Consciousness, called Shiva — and that everything that appears, including every individual self, is that one Consciousness freely taking form. Bondage is simply the forgetting of this; liberation is its recognition. The texts here are maps and methods for that recognition.
